# KodExplorerFlutter

由于习惯性的可以随时随地的查看自己的私有云文件，服务器安装了可道云，不过仅仅只能用浏览器登录查看，移动端的浏览器适配是惨不忍睹</br>

## 安装准备

基于可道云https://kodcloud.com/ 的第三方客户端，需要部署你的可道云服务器才可使用，登录需要服务器关闭验证码登录，可道云暂未适配方案。
地址就是你配置可道云的地址。

目前仅实现了登录退出功能、浏览文件列表、查看文件信息、复制外链的功能，后面会逐步更新</br>
