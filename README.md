## 安装准备


基于可道云https://kodcloud.com/ 的第三方客户端，需要部署你的可道云服务器才可使用，登录需要服务器关闭验证码登录，可道云暂未适配方案。
地址就是你配置可道云的地址。

项目使用的是Flutter编写，本人是一个android码农，所以ios方面涉及的不是很多，不过后续android端项目差不多完善了，再进行ios的更新

目前仅实现了登录退出功能、浏览文件列表、查看文件信息、复制外链、文件下载，后面会逐步更新</br>
